cppy by bl0b

some code aiming at statically inspecting C/C++ code, so as to be able to tell for instance where something is modified and who calls who.

Want to refactor some old code that you didn't even write ? means to get easily the general overview as well as the small details ? cppy should be for you.

nothing outside standard python modules.
